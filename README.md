The TPED Reader executable jar is a standalone, compiled version of the TPED Reader. You only need Java installed to run it, and none of the other files are necessary to use it. It is a portable program that you can cut/paste anywhere you want. 

The TPED Reader folder contains the source code that was written in Eclipse. 

This is an open source project that anyone is free to modify or use however they want as long as they provide credit to me (Jackson Monson, or JacquesCartier2 on github) for making the original program.
